flask-posts
===========================

### Requirements
* python 2.7
* virtualenv
* Flask
* sqlite3
* Flask-SQLAlchemy
* A github account

### Installation
**/!\ You need to fork this repository. See [How to submit your work?](#how-to-submit-your-work)**
```sh
git clone <repository>
cd flask-posts
virtualenv --no-site-packages venv
pip install -r requirements.txt
python app.py
